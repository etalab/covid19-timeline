Contributions d'Etalab pendant la pandémie Covid-19

Chronologie des contributions d'Etalab à l'information pendant la pandémie Covid-19.

2020-03-10

Publication d'un premier jeu de données de Santé Publique France

Etalab travaille avec Santé Publique France pour publier le jeu de données du nombre de cas confirmés au Covid-19.

Données : https://www.data.gouv.fr/fr/admin/dataset/5e689ada634f4177317e4820/

2020-03-12

Mise en ligne des chiffres clés concernant l'épidémie de COVID19 en France

Etalab contacte les auteurs du dépôt github.com/opencovid19-fr/data et publie un jeu de données issue du travail de volontaires contribuant à ce dépôt.

Données : https://www.data.gouv.fr/fr/datasets/chiffres-cles-concernant-lepidemie-de-covid19-en-france/

2020-03-16

Migration de data.gouv.fr vers udata 2.0

Etalab travaillait depuis longtemps sur la version 2.0 de udata. L'équipe effectue la migration vers cette nouvelle version.

2020-03-18

Ouverture des données de geodes.santepubliquefrance.fr

Etalab travaille avec Santé Publique France à l'ouverture des données mises à disposition sur le site https://geodes.santepubliquefrance.fr.

2020-03-19

Création d'une organisation pour Santé Publique France

Publication par Etalab des données SPF sur une nouvelle organisation SPF de data.gouv.fr (les fichiers publiés sont récupérés manuellement depuis geodes).

Provision d’un serveur data.gouv.fr pour réceptionner les données de SPF, mise en place de la sychronisation entre les fichiers de SPF et un répertoire de ce serveur (les données SPF ne sont pas encore disponibles).

2020-03-20

Amorce du travail sur l'algorithme d'orientation Covid-19

Capture d'écran du démonstration de l'algorithme d'orientation Covid-19

Héla Ghariani (de la Délégation numérique en Santé au ministère de la Santé) sollicite les développeurs de la DINUM pour de l'aide sur la publication de l'algorithme d'orientation Covid-19, algorithme déjà utilisé sur sur le site https://maladiecoronavirus.fr.

2020-03-31

Début des travaux sur une application d'orientation Covid-19

Échanges entre Etalab avec le Service d'information du gouvernement pour élaborer un questionnaire d'orientation similaire à celui qui est disponible sur https://maladiecoronavirus.fr, reprenant l'algorithme d'orientation tel qu'il est documenté sur ce dépôt.

2020-04-08

Mise en ligne de l'application d'orientation Covid-19

Page d'accueil du questionnaire d'orientation Covid-19

Le service d'information du gouvernement met en production une première version du questionnaire d'orientation développé par Etalab.

2020-04-13

Partage des données du questionnaire d'orientation avec l'institut Pasteur

Etalab partage les premières données reçues depuis le questionnaire d'orientation Covid-19 avec l'institut Pasteur.

2020-04-28

Point de contact avec les auteurs d'AlloCovid

Les auteurs d'AlloCovid se sont appuyés sur la documentation de l'algorithme d'orientation Covid-19.

2020-06-09

Publication des données des foires aux questions ministérielles

Etalab accompagne le service d'information du gouvernement dans la publication des contenus des foires aux questions publiées par les sites de l'État lors de la lutte contre la pandémie Covid-19.

Données : https://www.data.gouv.fr/fr/datasets/foires-aux-questions-publiees-par-les-administrations-dans-le-cadre-de-la-gestion-de-la-pandemie-covid-19/

2020-06-09

« Comment les administrations ont collaboré à l’ouverture des données du Coronavirus : le cas français »

Etalab publie un article de synthèse sur la façon dont les administrations ont collaboré à l'ouverture des données liées au Covid-19.

Source : https://www.etalab.gouv.fr/comment-les-administrations-ont-collabore-a-louverture-des-donnees-du-coronavirus-le-cas-francais
